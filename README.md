## HttpBin IP
> Useless package that saves your ip in a .txt file (:

### script installation:
```bash
git clone https://github.com/whyakari/httpbinip; cd httpbinip
```

### GO package installation:
```bash
go get github.com/whyakari/httpbinip@v0.5.1
```

### functions (until now):
 - save ip in a txt file
